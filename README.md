# Users-list

[Users-list](https://vvirk.github.io/users-list/).

## The simple React and Redux web app Blog.

## Features:
 - View users list
 - Create new user
 - Edit user
 - Delete user

## Workflow

### setup
```
git clone https://github.com/vvirk/users-list.git
cd users-list
npm install
```

### Run application
```
npm start
```
